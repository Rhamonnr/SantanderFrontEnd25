Tags H1 a H6

A tag h1 a h6 : São usadas para definir títulos e subtítulos de diferentes níveis.h1 é o mais importante e h6 o menos.

Exemplo:

H1

H2

H3

H4

H5
H6

Tag p

Define um parágrafo de texto. Pode-se usar quantas vezes for preciso no arquivo.

Tag mark

Destaca um trecho de texto, geralmente com uma cor de fundo amarela por padrão.

Exemplo:

MARK.

Tag small

Torna o texto menor, indicando que é menos importante ou para avisos legais.

Exemplo:

Hoje O DIA TA LINDO.

Tag i

Renderiza o texto em itálico, geralmente para indicar ênfase ou termos técnicos.

Exemplo:

Italico.

Tag u

Sublinha o texto, embora seu uso para links seja desencorajado (a menos que seja para um propósito específico).

Exemplo:

sublinhado

Tag strong

Indica texto com forte importância ou urgência, geralmente renderizado em negrito. Usado quando a palavra tem semantica.

Exemplo:

NEGRITO.

Tag ol

Cria uma lista ordenada (numerada).

Exemplo:

  1. Batata
  2. Frango
  3. Tomate

Tag ul

Cria uma lista não ordenada (com marcadores).

Exemplo:

  • Batata
  • Frango
  • Tomate

Tag li

Define um item de lista dentro de ol ou ul.

Exemplo a cima.

Tag a

Cria um hiperlink para outras páginas ou recursos. O atributo href especifica o URL e target="_blank" abre o link em uma nova aba.

Exemplo:

  • Google

Tag hr

Desenha uma linha horizontal na página, usada para separar blocos de conteúdo.

Exemplo:

Tag sub

Renderiza o texto como subscrito (abaixo da linha de base), comum em fórmulas químicas.

Exemplo:

Bom Dia gente.

Tag sup

Renderiza o texto como sobrescrito (acima da linha de base), comum em notações matemáticas.

Exemplo:

Bom Dia gente.

Tag blockquote

Indica que o texto é uma citação longa de outra fonte, geralmente com indentação. A tag cite dentro de um blockquote pode ser usada para o título da obra ou o nome do autor da citação.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Aut similique est quisquam, quidem impedit beatae animi. Magni aut, vel illo delectus, enim, fugiat doloremque ipsam quisquam ducimus labore molestias nesciunt. Todos os creditos ao Rhamon.

Envelopamento de Tag

Pressionando a tecla Ctrl + Shift + P, é possivel enveloparuma tag caso tenha esquecido.
